# Social Media Profile Images

Add your profile images here with the following filenames:

- `facebook.jpg` - Your Facebook profile image
- `instagram.jpg` - Your Instagram profile image
- `linkedin.jpg` - Your LinkedIn profile image
- `github.jpg` - Your GitHub profile image

The images should ideally be square (e.g., 200x200px or larger) and will be displayed as circular profile pictures on the Socials page.

If an image is not found, the social media icon will be displayed as a fallback.
